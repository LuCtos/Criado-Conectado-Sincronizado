# Criado-Conectado-Sincronizado
Exemplo de repositório para entregar na aula de Git/GitHub. Perkles e Falvojr

## Estrutura do Commit
### Assunto:
- Escreva em português ou em inglês dependendo da convenção da equipe
- Escreva no Imperativo! "Adiciona funcionalidade x"
### Corpo:
- Adicione detalhes ao commit
- Tente quebrar a linha em 75 caracteres
- identifique sua audiência - Quem vai ler
- Explique tudo
- Use Markdown
### Rodapé:
- Referencie assuntos relacionados
